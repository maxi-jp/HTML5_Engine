HTML5_Engine Documentation

A modular, object-oriented HTML5 game engine for web browsers, built with vanilla JavaScript, WebGL and the HTML5 Canvas API.

Engine and examples are actively hosted on GitHub Pages: https://maxi-jp.github.io/HTML5_Engine/

Examples

Explore the src/examples/ directory for practical demonstrations of the engine's capabilities:

  • Engine systems:
    • Renderers:
      • Canvas 2D API render test: Implements different renderer methods for primitives, sprites, animations...
      • WebGL render test: Same project but using the WebGL renderer.
    • Basic input system test
    • Input Actions Test: A dedicated example for the abstract input system (with Actions and Axis events).
    • Audio system test
    • Colliders test
    • Object pooling
    • Tileset (WIP)
    • Parallax (WIP)
  • Example games:
    • BrokeOut: Very basic breakout clone.
    • Floppy Derp: A Flappy Bird-like game.
    • Twin Stick Shooter
    • Twin Stick Shooter 2 (more complex)
    • Basic Tetris: Basic Tetris implementation.
    • Complex Tetris: Far more complex Tetris implementation with modern gameplay details.
    • Snake
    • Columns: Basic Columns like game (AI generated).
    • Puzzle Bobble: Basic Puzzle Bobble like game (almost full AI generated).
    • Pacmon (WIP): Implementation of the classic Pac-Man game.
  • UI and menus:
    • HTML + CSS main menu: Demonstrates basic menu implementation.
    • HTML + CSS main menu and game UI: A more elaborated example with UIs with interactions between the HTML menus and the game layers.
  • Box2D:
    • Box2D basic
    • Box2D basket
    • Box2D platformer (WIP)
  • Other example projects:
    • Mode7 (like SNES, sort of): Showcases the pseudo-3D Mode 7 effect (very WIP).
    • Lines intersection

Many more examples are available within the src/examples/ directory of the repository.
