HTML5_Engine Documentation

A modular, object-oriented HTML5 game engine for web browsers, built with vanilla JavaScript, WebGL and the HTML5 Canvas API.

Engine and examples are actively hosted on GitHub Pages: https://maxi-jp.github.io/HTML5_Engine/

Debugging Tools

To aid in development, the engine offers built-in debugging features:

  • Physics Debug Drawing: When using Box2D, you can enable debug drawing to visualize physics bodies, joints, and collision shapes.
  • FPS/Stats Overlay: An optional overlay can display frames per second (FPS) and other performance statistics.
