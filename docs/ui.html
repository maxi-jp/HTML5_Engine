HTML5_Engine Documentation

A modular, object-oriented HTML5 game engine for web browsers, built with vanilla JavaScript, WebGL and the HTML5 Canvas API.

Engine and examples are actively hosted on GitHub Pages: https://maxi-jp.github.io/HTML5_Engine/

UI & Menus

The engine encourages the use of standard HTML and CSS for building game user interfaces and menus. This allows you to leverage the full power of web development tools for creating rich and interactive UI elements.

You can overlay HTML elements on top of your canvas, or use the TextLabel and other UI-related GameObject's for in-game UI.
