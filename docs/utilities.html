HTML5_Engine Documentation

A modular, object-oriented HTML5 game engine for web browsers, built with vanilla JavaScript, WebGL and the HTML5 Canvas API.

Engine and examples are actively hosted on GitHub Pages: https://maxi-jp.github.io/HTML5_Engine/

Utilities

The engine includes various utility classes and functions to assist with common game development tasks:

  • utils_math.js: Provides helper functions for vector operations (Vector2), random number generation, and other mathematical calculations.
  • utils_classes.js: Contains fundamental classes like Color, TextLabel, and other drawing helpers.
  • Pool (from gameobjects.js): For efficient object pooling.

API Reference (utils_classes.js)

This section details some of the most common utility classes available in utils_classes.js.

Color Class

A class for creating and manipulating RGBA colors.

constructor(r, g, b, a=1)
Creates a color. R, G, and B values are expected to be between 0.0 and 1.0.
Color.FromRGB(r, g, b)
Static method to create a color from RGB values between 0 and 255.
Color.FromHex(hex)
Static method to create a color from a hex string (e.g., "#FF0000").
Color.Lerp(from, to, step)
Static method to linearly interpolate between two colors.
toString()
Returns the color as an `rgba(r,g,b,a)` string for use with the Canvas API.
Static Properties
Provides a wide range of predefined colors like Color.red, Color.blue, Color.transparent, etc.

Sprite Class

A helper class for drawing images (sprites) with transformations.

constructor(img, position, rotation, scale, alpha)
Creates a new sprite instance.
flipX / flipY
Boolean properties to flip the sprite horizontally or vertically.
Draw(renderer)
Draws the sprite with its current transformations (position, rotation, scale).
DrawSection(renderer, sx, sy, sw, sh)
Draws a specific rectangular section of the sprite's source image.

TextLabel Class

A helper class for drawing text to the canvas.

constructor(text, position, font, color, align, baseline, stroke, lineWidth)
Creates a new text label instance.
Properties: text, font, color
Getters and setters for the label's content and appearance.
Draw(renderer)
Draws the text to the canvas. For WebGL, it caches the text as a texture for performance.

API Reference (Colliders)

This section details the classes used for collision detection, also found in utils_classes.js.

CollisionManager Class

A static class that handles collision detection logic between different collider types.

CollisionManager.Check(colliderA, colliderB)
Checks for collision between two Collider objects. It automatically calls the correct specific check (e.g., circle-circle, circle-rect) based on the collider types.

Collider Base Class

The base class for all collision shapes. You typically use one of its subclasses.

OnCollisionEnter(myCollider, otherCollider)
A method on the associated GameObject that is called when a collision begins.
OnCollisionExit(myCollider, otherCollider)
A method on the associated GameObject that is called when a collision ends.
OnClick()
A method on the associated GameObject that is called when the collider is clicked.

RectangleCollider Class

An axis-aligned rectangular collider.

constructor(position, width, height, gameObject=null)
Creates a new rectangle collider. If a gameObject is passed, the collider will follow its position.

CircleCollider Class

A circular collider.

constructor(position, radius, gameObject=null)
Creates a new circle collider. If a gameObject is passed, the collider will follow its position.

PolygonCollider Class

A collider defined by a set of vertices, allowing for complex, non-rectangular shapes.

constructor(position, rotation, points, gameObject=null)
Creates a new polygon collider from an array of points (e.g., [{x: 0, y: -50}, ...]). The points are relative to the collider's position. If a gameObject is passed, the collider will follow its position and rotation.
